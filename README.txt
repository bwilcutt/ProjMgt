This folder contains a host of different documents pertaining to Software Project Management.  These documents were
originally designed to be held in a Document Control System by a company.  

Each document takes a Document Control Number (DCN).  This is a arbitrary but unrepeating number used to identify
your document.  Many companies have software that provides "guaranteed unrepeated numbers" for DCN, such as 
Atlassian.  The DCN number along with the document title (ie, "Project Widget Charter") are stored together and
commonly used in lists.

We use specific "tags" in all the documents that allow you to quickly adopt and "own" them.

[DCN} - Document control number
{Company Name} - Your company name
{...} - Hints given on what else may be needed for you to complete
